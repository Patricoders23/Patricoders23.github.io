LangChain: qué es y por qué importa en Oil & Gas

Mis primeros pasos con LangChain — qué son las chains, los prompts templates y cómo esto conecta con el análisis de datos de producción en pozos.

python langchain llm oil-gas

📌 Qué aprendí hoy

LangChain es un framework en Python que permite construir aplicaciones con LLMs (Large Language Models) de forma estructurada. La idea central es encadenar pasos: un prompt → un modelo → una respuesta → otra acción.

💡 Concepto clave

Una Chain en LangChain es como un pipeline: entrada de datos → procesamiento → salida. Exactamente como un workflow de ingeniería.

  • PromptTemplate — plantilla reutilizable para estructurar preguntas al modelo
  • LLMChain — conecta el prompt con el modelo y devuelve la respuesta
  • Memory — permite que el agente recuerde la conversación anterior
  • Tools — funciones externas que el agente puede usar (buscar, calcular, consultar BD)

⛽ Cómo lo aplico en Oil & Gas / Energía

En campo, cuando analizábamos datos de un pozo en Sacha, teníamos que consultar múltiples fuentes: registros de perforación, curvas de producción, datos de mud gas. Con LangChain se puede construir un agente que haga esas consultas automáticamente en lenguaje natural.

🔋 Caso de uso real

"¿Cuáles son los 3 pozos con mayor caída de producción este mes?" — una pregunta así podría resolverla un agente consultando directamente la base de datos de producción.

💻 Código de ejemplo

Mi primer chain en LangChain — una consulta simple sobre datos de producción:

Python 
# Instalar: pip install langchain openai

from langchain.prompts import PromptTemplate
from langchain.llms import OpenAI
from langchain.chains import LLMChain

# 1. Definir el prompt template
prompt = PromptTemplate(
    input_variables=["well_name", "production_data"],
    template="""
    Eres un ingeniero de petróleo analizando datos de producción.
    Analiza el pozo {well_name} con los siguientes datos: {production_data}
    Identifica si hay anomalías y sugiere acciones.
    """
)

# 2. Conectar con el modelo
llm = OpenAI(temperature=0.3)
chain = LLMChain(llm=llm, prompt=prompt)

# 3. Ejecutar con datos reales
result = chain.run(
    well_name="Sacha-101",
    production_data="Producción: 450 bbl/day, declining 15% MoM"
)

print(result)
⚠️ Nota importante

Para datos confidenciales de producción nunca usar OpenAI directamente. La semana próxima exploro cómo hacer esto mismo con Ollama en local.
